# Contributing

Gearbase is still in Phase 0 validation.

Current contribution rules:

- treat `gearbase-spec.md` as the product scope source
- treat `vendor/vara-eth-skills` and official Vara.eth docs as platform authority
- keep changes small and verifiable
- record platform findings in `docs/PLATFORM_NOTES.md`
- record spec drift in `docs/DEVIATIONS.md`
- update `docs/PROGRESS.md` when a phase milestone changes

Template contribution guidance will be expanded once the first room template is live.
